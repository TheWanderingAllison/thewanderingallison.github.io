解决无法在edge浏览器设置中使用搜索的问题

  • Toolbox
10 Jun, 2024

翻电脑文档发现之前的故障截图，顺便用中文关键词搜了下，结果时隔半年无论 Bing 还是 Google 都还是得不出靠谱结果，所以简单水一篇，或许能帮助到其他人。

去年莫名其妙地 edge 设置搜索功能无法使用，进入浏览器设置页面想搜比如 password 之类，只要在左上角搜索框输入字符，浏览器页面就会直接变空白，如下图。

网上也有人提出同样问题，像 MS Edge settings page goes blank when searching - Microsoft Community 和 edge浏览器设置中的搜索无法正常使用 - Microsoft Community 但里面的解决方案对我的浏览器都无效。最后是通过 Settings - Reset settings 重置设置（只会影响首页 pinned tabs 和需要重新启用插件，不会丢失其他信息）解决，之后没有再出现过类似问题。
