如何获取podcast的RSS

本篇所有探讨都基于网页以及搜索引擎，仅在 app 里才能获取的内容不配称之为互联网内容！ 虽然最近听播客的时间大大减少，但之前为了获取播客 RSS 的努力可是丝毫不少，来简单存个档吧。我日常使用的收听渠道是 Google Podcasts，英文世界的播客节目想要获取 RSS 都非常容易，大部分有官方网站，提供多渠道订阅方式和 RSS 链接，而中文播客届好多都无视 Google，不光在 Google Podcasts 页面搜不到，很多还根本不提供 RSS，找起来极其费劲。 1. 通过播客名获取 我最常用的是 Podbean ，直接搜索即可得到结果，有时同一播客有好几个 RSS 源，点进去挑个集数最全的就好。提供类似服务的还有 https://podcastrepublic.net 和 https://pod.link ，热门节目都可以搜到，冷门节目就得碰运气看谁家获取更快了。 2. 通过播客的 Apple Podcasts 链接获取 getRSSfeed 这个网站支持从 Apple Podcasts 、Google Podcasts 等多渠道解析 RSS。 有些播客通过第一种渠道无法检索到但已经提交到了 Apple Podcasts ，在已知链接的情况下可以用上文的网站获取。那么问题来了，Apple Podcasts 网页版是个摆设，也不支持站内搜索，没有他人分享无法直接获取节目链接，可以试着用 inurl:https://podcasts.apple.com/ 或者 site:https://podcasts.apple.com/ 这种方式找，但太冷门的播客此前根本没人访问过搜索引擎是不会收录的，于是变成了一个死循环。可能只能拜托身边用苹果的朋友分享一下了…… 3. 通过喜马拉雅 FM 专辑页面获取 还有些播客只在喜马拉雅更新，如果它是 host 在喜马拉雅上的话，那么在网站内搜索找到对应的专辑列表，网址通常是 https://www.ximalaya.com/album/XXXXXXXX ，在这串网址后加上 .xml 就是对应的 RSS 链接。 4. 小宇宙 这个流氓无解，除非主播在其他平台提交过，不然光凭小宇宙链接是搞不出 RSS 的，或者用 RSSHub 自建服务，可为了这狗屁平台我至于吗！

一些偷懒的豆瓣书影音同步流程

喊了一万遍豆瓣已死和逃离豆瓣，可它还是简中世界最重要的书影音数据集中地。从 2008 年用豆瓣，这些年几乎每一本读过的书、看过的电影都在豆瓣标记过，其他数据无所谓，这长达十几年的档案还是割舍不下。最早豆瓣有封禁苗头时我陆续用 豆坟 导出过几次数据，但这只是一堆 excel 表格，静止的数据并没有价值。期间用了几天 Goodreads，也试图用 neodb 取代豆瓣，但豆瓣最重要的一点：关注友邻的精神食粮动态，目前仍没有对手。因此我在挣扎之下处于半弃瓣状态，仅用它做标记，不再发言。 现在我的书影音档案在四个地方：豆瓣、notion、blog 以及 neodb。 豆瓣 纯手动，读完/看完后挨个标记。 Notion Notion 同步用的是在长毛象看到的 教程 。 原理是通过 GitHub Action 定期抓取用户的豆瓣 rss feed 来获取更新，缺点是每次只能抓取最近 10 条记录，此前数据只能手动导入（用豆坟或油猴插件一次性获取）。而豆坟导出的数据不含海报信息，这样一来从前的记录无法显示图片，总觉得略微遗憾。notion 本身的显示页面也不是很灵活，加载还很慢，得不停点击 load more 才会继续显示，我单纯是因为不需要手动就姑且当多个备份仓库了。 实现成果如下图。 Blog 同样是全自动，只需初始设定完毕后续就无需手动操作。 教程 豆瓣书影音同步 GitHub Action 。 原理是使用 GitHub Action 定期抓取用户豆瓣书影音记录，在个人 GitHub 仓库里生成对应 csv 文件。优点是第一次启用就能获取全部历史数据，后续的自动更新也没有上面的十条数量限制，还能自行编辑 csv 文件增加新条目，比如被豆瓣和谐的那些。缺点是导出的数据居然缺了 ISBN！等我好好学习完 GitHub Action 就自己重写一个（遥遥无期的 flag ）。 但获取数据只是第一步，将数据可视化还需要更多折腾。作者提供了两个途径，一是同步到 blog，二是同步到 notion 。 同步到 blog 具体就是往自己的静态博客里加几个网页，再写对应的 CSS ，让 csv 文件能够以图片墙形式显示，我抄的这个 「电影 / 阅读」页面，顶！ ， 略微做了点改动。

我选择保留偏见

混更新思路：翻开我的读书笔记。 几年前读特德姜的短篇集《降临》，《赏心悦目：审美干扰镜提案风波纪实》这篇我觉得很有意思，把当初的纸质笔记电子化一下存个档吧~ 总之就是基于人可以安装审美干扰镜去除外貌偏见的设定从而衍生出的一系列讨论。 读这篇时我反复摇摆，时而觉得去除审美干扰是个好主意，时而又觉得这相当于剥夺了一部分我的权利。 这篇的设置也非常有意思，多方人物多视角发言再加上不时的社会新闻播报，非常像现实生活中对热点事件的追踪。 而最终我的态度是：不会安装。 因为某种意义上，正是“我”的偏见、“我”的偏向和喜好塑造成了现在的“我”。哪怕一部分的我是由于当时的社会环境所造成，哪怕有失偏颇，但正是这所有的一切，才构成了当下的我。如果不能出于自由意志而做出决定，而是由科学、专家判定审美标准，又谈何自我呢？ 人是会变化的。人一生中会接触许许多多事物和理念，它们会从意想不到的方面去塑造人。就像流水击打卵石，恰似无所知觉，可每一次冲击都是雕琢的过程，水静止，卵石才定型。人也是这样吧，直到死亡的那一刻（或者停止思考的那一刻），才能盖棺论定，否则永远不要低估人改变的可能。 也正是这份流动性让每一个当下，每一份认知都变得独一无二。安装了审美干扰镜，相当于剥夺了人成长的一份乐趣。人会犯错，会走弯路，会先入为主，但是傲慢和偏见、嫉妒、自卑等等负面情绪造就了人的复杂性和迷人度，我不愿放弃这些。 而且，就算安装了外貌干扰镜又能如何呢？智商、性格、财富，等等等等，人类总会找到更多可判定、可比较的东西。更何况，谁又能保证这面镜子就是公正的？标准由谁制定？镜子又由谁制造？谁来安装，谁来调试，谁去后续跟踪维护数据？都是人，拥有偏见的人。因此永远不会存在绝对的公平。而在此基础上，让每个人都能根据自己喜好自由选择的社会，我觉得才是最大的公平。

不要这样做

只约束自己，不要求别人的互联网冲浪准则（不定期增补）。 我也不能完全做到，所以时常后悔，写下来算是给自己提个醒。 不使用长截图、二维码等方式分享内容，尽可能提供原网页链接。 不使用网络流行词和一切不知所谓的缩写、代称，不使用 emoji 表情代替文字。 不使用任何带有负面倾向的标签用语，包括但不限于“娇妻”、“婚驴/婚女”、“蝈蝻”、“偷国”、“丑国”等等等等，“小粉红”、“小留”、“老墨”、“阿三”也在此列。 不转发没有信源、只有截图的二手消息。 是的，忍不住吐槽前两天长毛象被转发了无数遍的 iOS 要审核相册敏感图片并且弹窗提示的嘟文。如此轻率地根据小红书一张截图就开始义愤填膺，拜托起码查证一下提供可靠消息来源吧？随便搜一下就发现这是从 2021 年就开始流传的消息，和那条小红书截图说的并不是一回事。iOS 15.2, iOS 17, Apple 官方介绍 , 哪怕我是个资深果黑也不高兴看到这种不负责任的传播方式。 不在公开场合聚众评价非公众人物。 我真的要管住自己的嘴！太想吐槽和一两个亲近的朋友私下说好了，也不要和网上半生不熟的人仅仅因为有共同讨厌对象就一起说坏话。 不使用站内短链以及各种带有追踪 id 的分享链接（每次看到别人发 B 站短链或者是带着一大串尾巴的长链接都心累）。 不要为小圈子沾沾自喜，不要使用内部黑话。 其实我挺理解的，圈层内的黑话会快速建立亲密感，但我不喜欢这样。就像读 LOTR 相关时，看到很多人用蘑菇丝指代魔苟斯，用摊牌、小熊、梅大、梅二（还是大梅、二梅？管他的，不重要）等等只有宝钻粉才能知道的梗。说实话，拖延了许久才读精灵宝钻，某种程度上也是因为这些粉丝给我的印象是抱团且排外（后续发现排外是真的，但圈内互掐也够离谱），让我望而生畏不愿加入。即使读完后我能 get 到这些称呼，我也不会使用。不想把交流变得如此困难。最近沉迷灌篮高手也是这样，发现许多人沉迷于给角色起昵称，只有看多了粉丝交流根据上下文才能推测出她们说的是谁。 不要在没有更多消息的情况下擅自揣度新闻事件当事人心理，还洋洋洒洒在社交媒体上发布小作文。 因为又看到一条没有任何专业相关背景就开始为刑事案件双方做心理画像的嘟，真是难以忍受。 不要主动询问对方年龄、职业、坐标等私人信息，不要不请自来给别人人生建议。管好我自己，以及，管好你自己。 是我太古板了吗？或许吧。 会少很多乐趣吗？可能吧。 但我总觉得，人还是需要一些原则和底线。

点击我的 阅读页面 即可跳转对应条目。 阿勒泰的角落 我的记忆出现了奇怪的偏差，一直觉得读过这本书，但看豆瓣并没标记过，笔记文档里却有篇写于几年前、只开了个头的感想。翻开后发现确实有许多似曾相识，不管了，再读一遍。李娟的好，很难用语言转述，无论何时读她的文字，总能收获到一种孤独中特有的宁静。 略摘几段。 我们还养了金鱼，每当和顾客讨价还价相持不下时，我们就请他们看金鱼。每次都成功地令他们大吃一惊，迅速转移注意力。在那时，当地人都还没见过真正的金鱼，只见过画片和电视上的。这样的精灵实在是这偏远荒寒地带最不可思议的梦一样的尤物——清洁的水和清洁的美艳在清洁的玻璃缸里妙曼地晃动、闪烁，透明的尾翼和双鳍像是透明的几抹色彩，缓缓晕染在水中，张开、收拢，携着音乐一般……而窗外风沙正厉，黄浪滚滚，天地间满是强硬和烦躁……这样，等他们回过神来，回头再谈价钱，口气往往会微妙地软下去许多。 出现在遥远的荒野之地的金鱼，在水中摇曳着的、朦胧的、似披着薄纱的生物，在当地人看来，会不会是梦一般的奇妙时刻？ 初生的小羊羔和初生的婴儿常常被一同放进彩漆摇篮里，挂在骆驼一侧。当骆驼走过身边，随手掀起摇篮上搭着的小毛毯，就有两颗小脑袋一起探出来。还有一个怀抱羊羔的老人，她看起来快要死了，但怀中的羊羔却又小又弱，犹是初生。她衣衫破损，神情安静。脚下一摊血淋淋的痕迹。她站在河边。河水轰鸣，冰雪初融。春天就要到了。  我一直在想，游牧地区的一只小羊羔一定会比其他地方的羊羔更幸运吧？会有着更为丰富、喜悦的生命内容。至少我所知道的羊，于牧人而言，不仅作为食物而存在，更是为了“不孤独”而存在似的。还有那些善良的，那些有希望的，那些温和的，那些正忍耐着的……我所能感觉到的这一切与羊羔有关的美德，以我无法说出的方式汇聚成海，浸渍山野，无处不在。我不相信这样的生活也能被改变，我不敢想象这样的生活方式有一天会消失。 还有冬天的羊羔们。 想到前阵子读的 放牧人生：湖区故事 里，人们也是如此珍重地对待初生的羊羔，但无可避免的，这种古老的生活方式确然在消失。 中亚行纪：土库曼斯坦、哈萨克斯坦、塔吉克斯坦、吉尔吉斯斯坦与乌兹别克斯坦之旅 卷首作者提到的土库曼斯坦的“地狱之门”。 4.5星。 不能单纯算是游记，更像是历史+人类学的混合体。虽然对于历史的描摹占了相当多篇幅（部分章节甚至有些泛滥），但瑕不掩瑜。作者从没强调过女性视角，许多细节却让读者自然意识到，只有女性才会在意这些。 这是作者的第一本书，可以看出明显的专业训练痕迹，是非常纯熟的非虚构写作手法。我也确实在发现她只是在每个国家停留几周时有些失望，但从另一方面来说，到达、进入这几个在很多人眼中神秘（且分不清）的中亚国家本就是一项值得称赞的举动。更别提她还曾见证到非常不可思议的场景（该国要屏蔽 YouTube 的地步），见证，在场，也是一种力量。
