我不知道的事：一些阅读flag

在这个既不是年末也不是月末的时间点写 flag 好像有点奇怪，只是因为我的电子书架和手边的纸质书都越堆越多，资源囤积规模以和我的阅读进度完全不匹配的速度疯涨。不想被无尽的书单淹没（还有想逃避写八九月的读书总结），列一下近来感兴趣的话题，尽量优先阅读专题范围内的书，不要见一本开一本（说是这么说但从来没能管住我的手）。 从小时候到二十来岁，我的兴趣似乎一直都是读故事，各种故事。人到中年，才发现对世界一无所知。近来越来越能从各种非虚构题材里获得快乐，希望未来几年的阅读方向就如标题所言：我不知道的事。想了解这个世界更多，不一定是人类的世界。 自然/生命/宇宙/科学 etc 自然 观鸟在自然相关书籍里占了大头，我的豆瓣友邻们也纷纷迷上了观鸟，我其实还没入门，只停留在观察家门口的鸟儿这一步，但这并不妨碍我享受阅读鸟类的书。今年读到的《游隼》、《远东冰原上的猫头鹰》、《羽毛》都超级喜欢。接下来打算继续阅读相关的一大箩筐，像科普类的《海鸟的哭泣》，研究人员的一手报告 A Most Remarkable Creature（据说这本比远东冰原上的猫头鹰还好看！）又或者自然文学类的 H is for Hawk、Vesper Flights 等等。 动物界的话，前阵子看了本《无敌蝇家》非常精彩，这本讲的是双翅目昆虫，接下来打算再读一本鞘翅目和鳞翅目昆虫的书。植物界，目前在读《蕨类植物的秘密生活》（好难！），还想读很多讲树的书，还有关于苔藓的《苔藓森林》。微生物界，那当然是顶流蘑菇啦（今年读的三本蘑菇书都不好，真生气，希望后面运气好一点）。 再有就是读完美国山川风物四记这套大部头。关于春天的《春满北国》我眼见着从春读到秋都还没结束，写得极好，但所有地名和生物名字都不熟悉，进度缓慢。 生命 生命究竟是怎么诞生的呢？对于“生命”的定义是确凿的吗？据说地球上生命的诞生是一个偶然，而第一个多细胞生命的出现是个黑洞，因此我产生了极大的好奇。而扩大生命的边界的话，地球的起源又在哪里？抱着这些疑问，我开始读尼克莱恩的《复杂生命的起源》，之前也读完了一本讲地球生命演化的 The Earth : a biography of life ，顺便还把《地球的故事》、《源自尘埃的世界》、《地球生命小史》等等加进了书单。 宇宙 想读一些如何观星、天文学知识入门又或者宇宙起源的书。之前读了一点关于天文学家的非虚构《初光：探寻宇宙的边缘》也很有趣，今年起码把这本给读完吧！ 科学 其实这里主要是指医学。我每天如此依赖我的身体，却对它如何运作一无所知。读库出了两套医学相关的文库本，“医学大神”和“人类与病”，各十四册，一百万字。了解医学怎样从混沌走向科学，了解人类如何发现疾病、控制疾病（或者无法控制）。恐惧源于无知，我想更多知道一些。 “医学大神”从 2022 年底开始读，上个月在图书馆借到了最后一册，结束了这个系列。“人类与病” 计划在年底前读完，是今年的另一个大计划了。 文明/宗教/艺术/建筑 etc 文明 目前主要的好奇对象是两个文明古国埃及和印度。列了一大堆书单，只是太不熟悉了，读起来很慢。 宗教 完全不信教，但我不排斥了解宗教。我甚至曾经读过很多佛经（完全忘了），也读过圣经、古兰经（没能读完）。更确切地说我想了解宗教的产生和它究竟怎样对人产生影响。对佛教的兴趣更大一些，基督教（以及天主教？）和伊斯兰教我没办法不带上有色眼镜看它们，只愿意走马观花地读一读。 艺术 非常遗憾地发现传统艺术在我们这一代似乎断了层，有多少人真的懂得欣赏书法和中国传统绘画？但某年去上博看董其昌展，入门处挂了颜真卿一副字，居然被震到想流眼泪，我都搞不明白那一刻冲动是从何而来。不求精深，只是想捡起一些被切断了的传承。 建筑 我对建筑并没有特别大兴趣，此处的主题阅读特指读库的“建筑史诗”系列，很喜欢王南的写作，之前读到的《塔窟东来》和《六朝遗石》都超级精彩。 摘一下这系列的简介： 读库出品“建筑史诗”系列丛书是由清华大学教师王南所写建筑普及类读本，计划分作四卷，每卷六册，共计二十四册。首卷为西方的西、东罗马及中国的汉唐（包括其间的魏晋南北朝时期）建筑，兼及影响中国至深的印度佛教建筑。次卷为西方中世纪及中国宋元建筑，附带深受中国影响的日本建筑，以及与西方建筑长期交流、融合的穆斯林建筑。三卷为西方文艺复兴时期及中国明代建筑。四卷为西方巴洛克、古典主义、新古典主义时期及中国清代建筑（包括神秘的藏传佛教建筑），以及对中西建筑历史文化的溯源、总结与阐发。 二十四册才出了十一册，翘首以盼 & 缓慢阅读 ing 。 舒适圈之内 就是纯纯消遣的小说啦！各种实验性的纯文学请走开，炫技的文字又或者过于个人的叙述时常让我觉得作者的存在感过强。我只想读简单的、畅快的好故事。 另外就是轻松愉快的各类随笔，属于不用动脑子就可获得的放松阅读体验。 舒适圈之外 各类漫画、绘本、图像小说。 可能认字比较早，家里大人也没有特地给我买过什么连环画（那时候没有绘本这种说法），总之小学一年级的我就在看只有字没有图的西游记了，等到我知道世界上还存在“漫画”这种东西的时候，完全习惯不了它的表达方式。但我也承认，图像或者绘画和文字有完全不一样的冲击力。也发现有很多给成人看的漫画，可以表达更复杂更晦涩的世界，而给孩子看的绘本，大人也一样可以欣赏它们的纯净美好。总之，想试试看不同的媒介。 其实感兴趣的话题还有很多，但不要太贪心了，后续我也肯定会偏离计划，就先到此为止吧！

记一次微小的成功

看各式教程我时常感到无可救药的挫败，不停地怀疑自己真的有这么愚蠢吗？虽然不能称聪明，起码基本的理解和学习能力我不应该缺少啊，为什么在看代码相关的内容总是如此地费劲，时时刻刻都在自我怀疑。 但也有成功解决问题的时刻，即使是非常微小的成功，也想记下来给自己一点鼓励。 好了言归正传。前阵子在微信读书上存档下了 《傲慢的堡垒》 这本书的 html 文件，按照惯例想转成 epub 时发现这本书的微信读书版本有个非常离谱的问题 —— 所有的 21 世纪日期都莫名其妙加了 100 年，从头到尾无一例外。第一章就是 “在2107 年……"，而豆瓣阅读的版本里数字都是对的，大概率应该不是出版社的问题吧。 总之，我无法容忍看到这些数字时将就着自行换算，所以打算纠正一下。肯定不能用查找替换把 “21” 换成 “20” 了事，毕竟还有 2021 年这种数字存在。 解决方案就是，在 calibre 里点击出查找替换功能，用 (21)(\d{2}) 这个 regex 表达式作为查找条件，然后 Mode 选择 Regex-function ，再点击 Function 旁边的 Create/edit，弹出编辑窗口，return 值改为 "20" + str(match.group(2)) 。命名这个 function 后返回上一界面，在 Function 下拉窗口找到自定义命令，查找，全部替换，完成！ （一共处理了 400 多处，在什么情况下才会把这么多数字都弄错啊！） 这肯定不是最优解，但管他呢！只要解决了我就是成功的！

(但较为真诚)的鸡汤。最精华在这个书名：一只鸟接着一只鸟。原以为是讲观鸟，后来才发现是写作指导书。一个早就认清自己没有写作天赋的人为什么还在坚持不懈地看写作建议？难道真的指望有什么书列出一二三四我就能灵光乍现写出神来之笔？说到底所有人给的建议最核心就还是这一条：坚持写下去。可能还有主打一个陪伴价值吧，让恨不得砸键盘或者撕笔记本的人知道不是自己一个人在苦熬。 书里的建议我也不是都赞同，比如这条：为你爱的人写作，为身边的人留下些什么。完全不同意。如果我可以称自己的文字为写作的话，那我完全是为自己而写。就算写身边的人，也只是因为我想记住他们，而不是给他们看。 承诺 啊我和布克奖真的合不来！读了好几本都觉得不对胃口。 作为自然文学来说不够好，而论知识性的部分也就前言我摘出来的这几段值得高亮一下。后面就全是作者的流水账日记，配上标本、照片和手绘也还勉强可以随便翻翻打发时间。还有这样说或许显得我很无情，但我真的不太关心作者作为抑郁症患者在一年间的心情碎片，每一章都读到“今天我很糟糕”、“今天我又提不起劲来”也让我很提不起劲来好吗！ 近年来，为了更好地理解“森林浴”，一些针对性的研究发现，在绿色大自然中散步，对身体许多系统都会直接产生积极影响。研究显示，在大自然中，特别是在树林中待过的受试者，血压会降低，压力激素皮质醇水平会下降，焦虑得到缓解，脉搏速度放慢。当人们在森林中度过一段时间后，身体中的交感神经系统活动水平会下降，针对压力的反击或逃跑反应也会消失。此外，还有一种叫作“自然杀伤细胞”的特殊白细胞，它们的活动会增加，这种细胞可以杀死被病毒感染的细胞和某些癌细胞。这些生化反应在受试者身上持续了一个月之久，而那些在城市环境中度过相同时间的受试者身上则没有发现这些变化。 为了防止受到病毒和细菌的感染，许多植物会产生挥发性物质和油脂，这些被统称为植物激素。研究“森林浴”的团队发现，吸入植物激素会对人体的免疫系统、内分泌系统、循环系统和神经系统产生类似的效果。这些油脂大部分没有非常强烈的气味，但能对人体产生效用。5月的树篱会散发出木香，这种香氛正是不同植物激素的混合体。当我们身处大自然的时候，会无意识地吸入它们。通过分析血清素的变化，可以发现更多线索。血清素是一种在大脑神经细胞之间传递信号的化合物，而这种神经递质的水平在抑郁症患者身体中较低。目前还不清楚血清素水平低究竟是情绪低落的原因还是其造成的后果，而且大脑中肯定还有其他机制参与情绪调节，但血清素和人类情绪之间似乎确实存在联系。与大自然互动会影响血清素水平，事实上，仅仅是身处户外就有效果：当阳光照射到皮肤或视网膜上时，会触发血清素释放；天气越晴朗，血清素释放的水平就越高。11月至3月期间，北半球日照水平较低，会导致一些人患上冬季抑郁症（也叫季节性情感障碍）。 还有一个更令人惊讶的方式会影响血清素的水平，那就是和泥土亲密接触。有时人会接触到土壤中的益生菌。以母牛分枝杆菌为例，这种细菌的细胞壁上有一种蛋白质，可以触发人类大脑中特定的神经细胞释放血清素。这么看来，修剪草坪的益处可不仅仅是让草坪变得整齐了。最后一点，当我们进行诸如散步这样的轻度运动时，血液中会释放内啡肽。这种神经递质可以减少疼痛，并带来轻微的欣快感：一种温和、自然的兴奋。叠加阳光、植物激素和土壤中益生菌的作用，于是当我们在花园、田野和树林中散步时，就好像走进了一个无形的天然药箱。 插播一张书里的椋鸟图片。 弃掉的 菌物志 看来七月不宜读蘑菇——这是第三本观感不佳的蘑菇书了！我甚至没能读完。无法忍受这种拟人风格，语气上刻意卖萌，真正涉及到科普的部分却又没办法写的生动活泼，只能生硬地搬运专有名词，实在读不下去。 激情耗尽 文风活像三四十年前的香港言情，拉拉杂杂半天也不知道作者到底要讲啥。 绿灯 三江源的猛禽都快被电线电死了？ 最近闹得沸沸扬扬的三江源保护区大量猛禽触电身亡事件，总算有了一个看起来比较理性中立的解读。顺便又看到另一篇，能在电线上排排站的鸟儿，也会触电？ ，讲电线和鸟类保护的。希望这次的猛禽触电事件能够有一个好的调查结果和更好的保护方案吧。 戴斯蒙·麦卡锡：成就斐然的失败者 不知怎么，看到天赋如此卓绝的人到最后并没有写出传统意义上的伟大作品而是毕生都在写文艺评论居然奇异地让我好受了点？文章里说，“The best is the enemy of the good.” 但我的问题是我和 good 都还差得远哪！每次想写点什么都痛苦万分地挣扎我为什么要写？为什么写得这么烂？因此一旦看到不止我在受折磨就获得了一点点安慰。 The Double Education of My Twins’ Chinese School 何伟的新文。看他讲女儿的数学题让我笑得不行（是的，中国的数学题就是想法设法在给人设陷阱），但从中美教育又引出他和妻子双方家族历史，这一段真是妙。他的外祖父，和他妻子的祖父，一个美国人去意大利接受本笃会的教育梦想去中国传教，一个中国人去了美国从事矿业最后又死在了东北，历史和现在，这种对比真是让人战栗。 他的材料组织能力和叙述角度确实强过许多非虚构作者，哪怕我很清楚地知道他的文章其实也是一种纯熟的技巧，但总有许多地方打动我。 回形针解散两年了，他们怎么样了？ 在方可成的新闻实验室看到这篇，还挺意外的。我一直记得回形针这个自媒体，但我既没看过他们任何视频，也对他们为何被封一头雾水。确切地说，我是直到他们在被封前夕（也就是被大范围讨论政治立场问题时）才听说他们的名字。在这个地方，可能只要任何非官方渠道掌握了一定的舆论，就会引起警惕，迟早会被消声。 想起之前我关注的另一位博主，科学松鼠会主笔之一的 ent，因为对 731 部队是否采用细菌战出言不慎被大范围网暴再至被强制注销各平台账号。我刷微博不勤，那几天只是非常奇怪为什么他的微博下面评论数量暴涨——一般科学类的百十来条就算多了，可那阵子不管他发什么都有近千条评论，我以为他终于红了，没想到是另一种意义上的红。是的，我迟钝到直到他发道歉信才搞清楚发生了什么，也是那个时候点开之前的评论区我才发现全是谩骂。很喜欢看他的 ImagineNature 专栏，时隔一年多，这个公众号终于又更新了一篇文章，我松了口气。能恢复写作，说明从旧日的风波里多少恢复过来了吧。不知是否是因为这个公众号没有直接挂名 ent，亦或者是腾讯网开了一面，总之，还是保留了一条可关注的渠道。我对这次事件里网民的义愤填膺和官方的无限纵容偏颇感到胆战心惊。 Top 10 WEIRD CATERPILLARS You’ve Never Heard Of!!! 恐虫者慎入。但我真是大开眼界，好多虫子长得太过外星人甚至没有可怕的感觉只让我觉得 “蛤，还能长这样？”，尤其是 dragon caterpillar 还怪可爱的。

如何获取podcast的RSS

本篇所有探讨都基于网页以及搜索引擎，仅在 app 里才能获取的内容不配称之为互联网内容！ 虽然最近听播客的时间大大减少，但之前为了获取播客 RSS 的努力可是丝毫不少，来简单存个档吧。我日常使用的收听渠道是 Google Podcasts，英文世界的播客节目想要获取 RSS 都非常容易，大部分有官方网站，提供多渠道订阅方式和 RSS 链接，而中文播客届好多都无视 Google，不光在 Google Podcasts 页面搜不到，很多还根本不提供 RSS，找起来极其费劲。 1. 通过播客名获取 我最常用的是 Podbean ，直接搜索即可得到结果，有时同一播客有好几个 RSS 源，点进去挑个集数最全的就好。提供类似服务的还有 https://podcastrepublic.net 和 https://pod.link ，热门节目都可以搜到，冷门节目就得碰运气看谁家获取更快了。 2. 通过播客的 Apple Podcasts 链接获取 getRSSfeed 这个网站支持从 Apple Podcasts 、Google Podcasts 等多渠道解析 RSS。 有些播客通过第一种渠道无法检索到但已经提交到了 Apple Podcasts ，在已知链接的情况下可以用上文的网站获取。那么问题来了，Apple Podcasts 网页版是个摆设，也不支持站内搜索，没有他人分享无法直接获取节目链接，可以试着用 inurl:https://podcasts.apple.com/ 或者 site:https://podcasts.apple.com/ 这种方式找，但太冷门的播客此前根本没人访问过搜索引擎是不会收录的，于是变成了一个死循环。可能只能拜托身边用苹果的朋友分享一下了…… 3. 通过喜马拉雅 FM 专辑页面获取 还有些播客只在喜马拉雅更新，如果它是 host 在喜马拉雅上的话，那么在网站内搜索找到对应的专辑列表，网址通常是 https://www.ximalaya.com/album/XXXXXXXX ，在这串网址后加上 .xml 就是对应的 RSS 链接。 4. 小宇宙 这个流氓无解，除非主播在其他平台提交过，不然光凭小宇宙链接是搞不出 RSS 的，或者用 RSSHub 自建服务，可为了这狗屁平台我至于吗！

言简意赅的几款方便食品点评

可能没什么参考价值的一些方便食品点评。先说我的口味：淡，非常淡，水煮青菜不放任何调料都可以面不改色吃下去。不吃辣（虽然硬吃也能吃一点），不爱一切重口食品（包括但不限于火锅、干锅、酸菜鱼、烧烤等等），所以我很有自知之明从不和别人约饭，几乎没几个人能和我吃到一块去。 阿宽红油面皮、麻酱面皮 要说多好吃也不至于，但是有时就是很想吃凉皮又没得买/懒得出门，就拿这个做平替。撕开包装倒入开水，五分钟就能吃，非常省事。 红油面皮辣得我涕泪齐下，每次小心翼翼只能倒一丁丁点辣油进去，还经常手抖倒多，然后狂喝水。后来发现原来有麻酱味的欣喜若狂，终于不用颤颤巍巍倒红油包了！ 阿宽甜水面 我的感想：甜，腻，撑。 但我的苏南 & 上海朋友都喜欢吃这个，可能我和甜口的面八字不合。 农心辛拉面 没人不知道这牌子吧！ 我吃过三个口味：辣白菜、香菇牛肉、芝士辣白菜。 最经典的还是辣白菜，不辣（我在说什么？）。香菇牛肉，有点辣。芝士味，超级无敌腻。 总之，还是乖乖买经典款吧！认准下面这个包装。 LIAN Pho Bo 越南河粉 朋友在 Costco 买了一箱，分了点给我尝尝，好难吃啊，全是味精的味道…… 幸好不是我买的。 随便搜了张淘宝代购的图。长这样。 云品鲜过桥米线 在长毛象偶然看到安利，说这牌子米线很好吃，通过了云南人认证的那种。我立刻下单把四个口味都买了。只吃了素食菌汤和菊花鸡肉这两个味道，先说优点，如上图所示，料包非常丰富，调味也比较克制不会过齁，没有讨厌的味精味。 而我不会回购的几个理由： 包装太多，让我觉得非常不环保。 米线是双人份调味料却只有一份，一个人煮时分量很尴尬。 虽然口味不算很油腻但是用完的锅和碗都会有挺厚的一层油边，我心理上过不去。 配料表相对来说已经算干净，可四个口味无一例外都是用的精炼植物油。 不过我把辣味的分给朋友之后对方光速吃完并且朝我索要链接，并且说比他买的另一款价格更高的松茸鸡汤米线好吃多了！ 安食六和兰州拉面 居然连拉州拉面都有速食装了！免煮，直接开水冲泡即可。确实是店里的味道，但杯面的分量超级少，完全吃不饱啊…… 但我发现这牌子有好多种包装，并且还有需水煮的袋装面，下次试试看。 拉面说 懒得找图了，一个曾经网红过的牌子，以料包丰盛出名，但不打折买的话完全不划算，而且除了番茄味的另外几款都好腻。 蓝色烟囱意面 又一个算是网红的牌子？朋友激情下单了全部口味但又不爱吃意面，最后几乎全投喂给我了。蟹黄味的咸且腻，其他的没什么记忆点。要是碰上打折可以囤点，起码不用自己开伙。 总结，还是老实自己做吧，我这个挑剔鬼真的不适合吃现成的！

一些偷懒的豆瓣书影音同步流程

喊了一万遍豆瓣已死和逃离豆瓣，可它还是简中世界最重要的书影音数据集中地。从 2008 年用豆瓣，这些年几乎每一本读过的书、看过的电影都在豆瓣标记过，其他数据无所谓，这长达十几年的档案还是割舍不下。最早豆瓣有封禁苗头时我陆续用 豆坟 导出过几次数据，但这只是一堆 excel 表格，静止的数据并没有价值。期间用了几天 Goodreads，也试图用 neodb 取代豆瓣，但豆瓣最重要的一点：关注友邻的精神食粮动态，目前仍没有对手。因此我在挣扎之下处于半弃瓣状态，仅用它做标记，不再发言。 现在我的书影音档案在四个地方：豆瓣、notion、blog 以及 neodb。 豆瓣 纯手动，读完/看完后挨个标记。 Notion Notion 同步用的是在长毛象看到的 教程 。 原理是通过 GitHub Action 定期抓取用户的豆瓣 rss feed 来获取更新，缺点是每次只能抓取最近 10 条记录，此前数据只能手动导入（用豆坟或油猴插件一次性获取）。而豆坟导出的数据不含海报信息，这样一来从前的记录无法显示图片，总觉得略微遗憾。notion 本身的显示页面也不是很灵活，加载还很慢，得不停点击 load more 才会继续显示，我单纯是因为不需要手动就姑且当多个备份仓库了。 实现成果如下图。 Blog 同样是全自动，只需初始设定完毕后续就无需手动操作。 教程 豆瓣书影音同步 GitHub Action 。 原理是使用 GitHub Action 定期抓取用户豆瓣书影音记录，在个人 GitHub 仓库里生成对应 csv 文件。优点是第一次启用就能获取全部历史数据，后续的自动更新也没有上面的十条数量限制，还能自行编辑 csv 文件增加新条目，比如被豆瓣和谐的那些。缺点是导出的数据居然缺了 ISBN！等我好好学习完 GitHub Action 就自己重写一个（遥遥无期的 flag ）。 但获取数据只是第一步，将数据可视化还需要更多折腾。作者提供了两个途径，一是同步到 blog，二是同步到 notion 。 同步到 blog 具体就是往自己的静态博客里加几个网页，再写对应的 CSS ，让 csv 文件能够以图片墙形式显示，我抄的这个 「电影 / 阅读」页面，顶！ ， 略微做了点改动。
