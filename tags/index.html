点击我的 阅读页面 即可跳转对应条目。 一只鸟接着一只鸟：关于写作与人生的建议 老生常谈 (但较为真诚)的鸡汤。最精华在这个书名：一只鸟接着一只鸟。原以为是讲观鸟，后来才发现是写作指导书。一个早就认清自己没有写作天赋的人为什么还在坚持不懈地看写作建议？难道真的指望有什么书列出一二三四我就能灵光乍现写出神来之笔？而且说到底所有人给的建议最核心就还是这一条：坚持写下去。 书里的建议我也不是都赞同，比如这条：为你爱的人写作，为身边的人留下些什么。完全不同意。如果我可以称自己的文字为写作的话，那我完全是为自己而写。就算写身边的人，也只是因为我想记住他们，而不是给他们看。 承诺 啊我和布克奖真的合不来！读了好几本都觉得不对胃口。 写得不算糟糕，但我不喜欢，拖了四个月终于了结。视角和人称的转换不影响我理解，但是让我怀疑这是否有必要——尤其是莫名其妙就从第三人称变成第二人称，这到底有什么意义？说是南非家族史，可惜又是白人的家族史，黑人女佣的存在感低到接近于无，如果我想了解南非种族隔离的历史，那我宁愿听黑人的讲述。 铸剑 鲁迅经典小说的图像化。我觉得这样的尝试值得鼓励，但是……这个成品实在不能令人满意。图像大大削弱了原文的冲击力。 献给爱与子宫的花束：夜女郎的母女论 读完《始于极限》对铃木凉美实在是有些好奇，于是找来她的这本随笔读了。 波波尔乌：插图重述版 蘑菇图鉴 只有图没有鉴，完全没有科属说明，整本书只有原图鉴的模糊不清拉丁文和日文，是指望读者懂拉丁语还是懂日语啊？蘑菇画是很美，可如果出版社想做成艺术图册至少把开本做大些，并且起码能够平摊吧？这个版本别说平摊，开合角度想要稍大都要费九牛二虎之力使劲儿压书脊。胶装的方式我估计也根本经不住几次压就要分崩离析。印刷质量也一般般，用的还是会反光的光面纸，翻起来真的好恼火。只能庆幸我在多抓鱼买的二手书并且立刻卖回去了。 吃货的生物学修养：脂肪、糖和代谢病的科学传奇 太一般了，俗套的公众号写作。隔几句话就插个不好笑的俏皮话想搔读者的痒，可能就是给看不下长文的公众号受众看的吧。 菌中毒 在随机波动那里种草却踩雷的第二本蘑菇书了。目前来说初次种草来源是她们的文艺作品就没有特别对我胃口的，有些只是喜好不同，有些则是真的质量问题。比如这本。 我看标题本以为是菌中（一声）毒，也就是讲蘑菇里的毒素，翻开才发现断句应该是菌中（四声）毒。要不是曾孝濂老先生的插图我可能会怒打两星…… 流水账一般的记录，内容一句话概括就是，“我和我认识的云南名人们”，和蘑菇的关系大概仅限于“我记得那个谁谁吃了啥啥菌子”吧。真是白瞎了这么美的封面和那么美的插图！ 关灯就睡觉：这样治疗失眠更有效 没有废话的实用指南。 初版于 1999 年，里面很多知识隔了 20 多年还被播客又翻出来作为新知传播（并大卖），yes 说的就是非常火热的 Huberman Lab。这让我更加觉得播客挺没必要的，至少在信息密度上真是完全不能满足我。 打算回头好好整理摘录再发篇笔记（希望我不会鸽掉）。 大自然治好了我的抑郁症：365天荒野疗愈手记 作为自然文学来说不够好，而论知识性的部分也就前言我摘出来的这几段值得高亮一下。后面就全是作者的流水账日记，配上标本、照片和手绘也还勉强可以随便翻翻打发时间。还有这样说或许显得我很无情，但我真的不太关心作者作为抑郁症患者在一年间的心情碎片，每一章都读到“今天我很糟糕”、“今天我又提不起劲来”也让我很提不起劲来好吗！ 近年来，为了更好地理解“森林浴”，一些针对性的研究发现，在绿色大自然中散步，对身体许多系统都会直接产生积极影响。研究显示，在大自然中，特别是在树林中待过的受试者，血压会降低，压力激素皮质醇水平会下降，焦虑得到缓解，脉搏速度放慢。当人们在森林中度过一段时间后，身体中的交感神经系统活动水平会下降，针对压力的反击或逃跑反应也会消失。此外，还有一种叫作“自然杀伤细胞”的特殊白细胞，它们的活动会增加，这种细胞可以杀死被病毒感染的细胞和某些癌细胞。这些生化反应在受试者身上持续了一个月之久，而那些在城市环境中度过相同时间的受试者身上则没有发现这些变化。 为了防止受到病毒和细菌的感染，许多植物会产生挥发性物质和油脂，这些被统称为植物激素。研究“森林浴”的团队发现，吸入植物激素会对人体的免疫系统、内分泌系统、循环系统和神经系统产生类似的效果。这些油脂大部分没有非常强烈的气味，但能对人体产生效用。5月的树篱会散发出木香，这种香氛正是不同植物激素的混合体。当我们身处大自然的时候，会无意识地吸入它们。通过分析血清素的变化，可以发现更多线索。血清素是一种在大脑神经细胞之间传递信号的化合物，而这种神经递质的水平在抑郁症患者身体中较低。目前还不清楚血清素水平低究竟是情绪低落的原因还是其造成的后果，而且大脑中肯定还有其他机制参与情绪调节，但血清素和人类情绪之间似乎确实存在联系。与大自然互动会影响血清素水平，事实上，仅仅是身处户外就有效果：当阳光照射到皮肤或视网膜上时，会触发血清素释放；天气越晴朗，血清素释放的水平就越高。11月至3月期间，北半球日照水平较低，会导致一些人患上冬季抑郁症（也叫季节性情感障碍）。 还有一个更令人惊讶的方式会影响血清素的水平，那就是和泥土亲密接触。有时人会接触到土壤中的益生菌。以母牛分枝杆菌为例，这种细菌的细胞壁上有一种蛋白质，可以触发人类大脑中特定的神经细胞释放血清素。这么看来，修剪草坪的益处可不仅仅是让草坪变得整齐了。最后一点，当我们进行诸如散步这样的轻度运动时，血液中会释放内啡肽。这种神经递质可以减少疼痛，并带来轻微的欣快感：一种温和、自然的兴奋。叠加阳光、植物激素和土壤中益生菌的作用，于是当我们在花园、田野和树林中散步时，就好像走进了一个无形的天然药箱。 看来七月不宜读蘑菇——这是第三本蘑菇书了！但我甚至没能读完。无法忍受不了这种拟人风格，语气上是刻意卖萌，真正涉及到科普的部分却又没办法写的生动活泼，弃了。 激情耗尽 文风活像三四十年前的香港言情，拉拉杂杂半天也不知道作者想讲啥。 绿灯 小农意识要不得——嗯我在说因为看到微信读书号称独家首发且免费就好奇点开看的自己，这破碎又混乱的描述到底是要什么？估计只有死忠粉才能忍着读完吧。看了10%弃了。

人类学家在多年后发现自己田野调查期间一直被秘密警察当作间谍监视，国安局里关于她的材料多达几千页，而申请档案解密后，以这些材料为原始资料，她再次回访、溯源最初的访谈对象、线人、告密者甚至国安局军官，最终完成了这本书。 我没有很喜欢它，甚至它让我再次对人类学的存在意义产生怀疑。作者一开头做田野的毫无自觉也让我只想评价她有一种“清澈的愚蠢”，但后期她阅读自己的档案过程中数次的不安、沮丧和怀疑很真实，那时有打动到我。 四星胜在题材，在威权社会，人人皆可能是告密者的环境里，人和人的关系究竟会是怎样的存在？70 年代依靠人力的罗马尼亚似乎也不能给现在被智能手机、电脑和摄像头包围的我们答案。 她的空难和她 1971年秘鲁航空空难中，作者从三千米高空坠落，又在雨林里独自跋涉了11天，是这起空难唯一的生还者。经历了这样的灾难之后，她并没有对雨林产生畏惧，甚至在数十年后重回秘鲁，试图为当地建立生态保护区。 “奇迹女孩”故事养活了无数媒体，她却直到 30 年后才首度执笔讲述。整本书文学性稍逊一筹，但作者经历就足以让人屏息。不仅仅是关于空难，也是整段人生的回顾，我很喜欢她冷静理智的笔触。 而书中没有展开详述的同为生物学家的父母生平也极为传奇。她父亲在二战后花费数年从德国到达秘鲁从事研究的过程，堪比西天取经。她母亲在那个职业女性都不多见的时代，是世界知名的鸟类学家。这样的父母，才会有这样的女儿。 我现在知道了，我父母，尤其是我母亲，可能也有这种强烈的感受。我能用所有感官来感受雨林，体会动物和植物世界的无限多样性，想象生物对环境精巧的适应，觉察大自然中细微的色彩变化，感受有时候像外套一样包裹着我的、至今依然让我牵挂的各种声音，和雨林中的气味、昏暗的黄绿色光线，以及温暖又潮湿的空气。我觉得自己好像变成了一个巨大的、包罗万象的生命体的一部分，对一切都那么熟悉，可对新事物依然充满好奇。这种不断能发现新事物的可能性，正是我们科学家们对热带雨林的生物感到惊奇的原因。尤其是在潘瓜纳，我们已经在这里做了四十年研究，依然有很多未解之谜。 另外书中提及她曾被赫尔佐格说动，拍过部记录片 Wings of Hope，不看电影的我想了半天才意识到这个名字如此熟悉是因为他在我非常喜欢的《游隼》推荐语里出现过。（世界果然是相连的！） 清晨鱼市与深夜书桌 好想对大陆引进后的这个鸡汤名字翻白眼。台版原名《伪鱼贩指南》，父亲欠下巨额赌债，作者中断学业，接下了家中经营三代的贩鱼生意。写鱼，更写人。随笔集，各篇水准参差不齐，但我愿意给首部作品一些宽容。同为台湾新人作者，《老派少女购物路线》是中产阶层的优裕闲适，圆融却无趣。这本粗粝，但真实。全本最佳还是写祖父的『弓鱼』一篇，尖锐到足以隔空刺痛读者。（但鱼贩备孕的故事就不必讲了吧！真的全无兴趣！） 给我翅膀 三星半。原以为是讲鸟类的纪实文学没仔细分辨就兴高采烈开读了，越看越不对劲去翻介绍才发现是小说（扶额）。 网瘾少年和身为鸟类学家的父亲一起帮助濒危白额雁开辟全新迁徙路线的故事。怎么说呢，如果没有父母爱情的烂俗桥段的话，我会给这本虽然不太现实但是十分可爱的书打四星，遗憾的是非要追求大团圆结局让它沦为了三流文本，是个适合拍成合家欢电影的故事，作为文字来说还是欠缺了点力量。 但读到这样的段落时，我也和那个忘记了游戏世界的小少年一样，是沉醉的。 天空蓝得那样浓烈，那样纯净，美得让人屏息。上升就像是反方向的潜水，慢慢地向上深入。 这天晚上，在黄昏时分，他们来到瞭望塔上。似乎上天也知道，这是他们在卡马尔格的最后一夜，于是让这里蔓延着近乎超自然的温柔气氛。夕阳的光辉洒在潟湖上，闪耀着玫瑰金色的光芒。这一片宁静是如此完美，身处其中，可以听见大地原本的细碎声响，听见芦苇丛中的风声，听见一只啮齿动物在灌木丛中经过的声音，或是远远传来的一声鸟鸣。 有同名电影，拍的很不错，影像完美地呈现了这一幕。而看到雁群跟随他们的超轻型飞机飞起的那一刻，我无法控制地在屏幕前热泪盈眶。自然的力量无法抗拒。 书里还有几段也让我颇多感慨，再长篇累牍地摘录似乎没有必要，真希望有读过的朋友能和我一起聊聊啊。 但到底什么是最荒谬的？是想要训练一群鸟穿越整个欧洲，以求拯救这个物种，还是借口为时已晚，事不关己地看着世界走向毁灭？ 还有，为鸟儿领航这件事是真实的！法国科学家 Christian Moullec 已经和鸟儿一起飞行了 20 多年，搜了篇时间最近的相关 文章。 A Bed for the Winter 翻电脑存货翻到这本，DK 童书系列，也不知什么时候下的，全是图片，给 5~7 岁的小孩认字用的，然而我居然还不是所有单词都认识，哭🥲。 东京八平米 写不动了…… 如何获取podcast的RSS

本篇所有探讨都基于网页以及搜索引擎，仅在 app 里才能获取的内容不配称之为互联网内容！ 虽然最近听播客的时间大大减少，但之前为了获取播客 RSS 的努力可是丝毫不少，来简单存个档吧。我日常使用的收听渠道是 Google Podcasts，英文世界的播客节目想要获取 RSS 都非常容易，大部分有官方网站，提供多渠道订阅方式和 RSS 链接，而中文播客届好多都无视 Google，不光在 Google Podcasts 页面搜不到，很多还根本不提供 RSS，找起来极其费劲。 1. 通过播客名获取 我最常用的是 Podbean ，直接搜索即可得到结果，有时同一播客有好几个 RSS 源，点进去挑个集数最全的就好。提供类似服务的还有 https://podcastrepublic.net 和 https://pod.link ，热门节目都可以搜到，冷门节目就得碰运气看谁家获取更快了。 2. 通过播客的 Apple Podcasts 链接获取 getRSSfeed 这个网站支持从 Apple Podcasts 、Google Podcasts 等多渠道解析 RSS。 有些播客通过第一种渠道无法检索到但已经提交到了 Apple Podcasts ，在已知链接的情况下可以用上文的网站获取。那么问题来了，Apple Podcasts 网页版是个摆设，也不支持站内搜索，没有他人分享无法直接获取节目链接，可以试着用 inurl:https://podcasts.apple.com/ 或者 site:https://podcasts.apple.com/ 这种方式找，但太冷门的播客此前根本没人访问过搜索引擎是不会收录的，于是变成了一个死循环。可能只能拜托身边用苹果的朋友分享一下了…… 3. 通过喜马拉雅 FM 专辑页面获取 还有些播客只在喜马拉雅更新，如果它是 host 在喜马拉雅上的话，那么在网站内搜索找到对应的专辑列表，网址通常是 https://www.ximalaya.com/album/XXXXXXXX ，在这串网址后加上 .xml 就是对应的 RSS 链接。 4. 小宇宙 这个流氓无解，除非主播在其他平台提交过，不然光凭小宇宙链接是搞不出 RSS 的，或者用 RSSHub 自建服务，可为了这狗屁平台我至于吗！

可能没什么参考价值的一些方便食品点评。先说我的口味：淡，非常淡，水煮青菜不放任何调料都可以面不改色吃下去。不吃辣（虽然硬吃也能吃一点），不爱一切重口食品（包括但不限于火锅、干锅、酸菜鱼、烧烤等等），所以我很有自知之明从不和别人约饭，几乎没几个人能和我吃到一块去。 阿宽红油面皮、麻酱面皮 要说多好吃也不至于，但是有时就是很想吃凉皮又没得买/懒得出门，就拿这个做平替。撕开包装倒入开水，五分钟就能吃，非常省事。 红油面皮辣得我涕泪齐下，每次小心翼翼只能倒一丁丁点辣油进去，还经常手抖倒多，然后狂喝水。后来发现原来有麻酱味的欣喜若狂，终于不用颤颤巍巍倒红油包了！ 阿宽甜水面 我的感想：甜，腻，撑。 但我的苏南 & 上海朋友都喜欢吃这个，可能我和甜口的面八字不合。 农心辛拉面 没人不知道这牌子吧！ 我吃过三个口味：辣白菜、香菇牛肉、芝士辣白菜。 最经典的还是辣白菜，不辣（我在说什么？）。香菇牛肉，有点辣。芝士味，超级无敌腻。 总之，还是乖乖买经典款吧！认准下面这个包装。 LIAN Pho Bo 越南河粉 朋友在 Costco 买了一箱，分了点给我尝尝，好难吃啊，全是味精的味道…… 幸好不是我买的。 随便搜了张淘宝代购的图。长这样。 云品鲜过桥米线 在长毛象偶然看到安利，说这牌子米线很好吃，通过了云南人认证的那种。我立刻下单把四个口味都买了。只吃了素食菌汤和菊花鸡肉这两个味道，先说优点，如上图所示，料包非常丰富，调味也比较克制不会过齁，没有讨厌的味精味。 而我不会回购的几个理由： 包装太多，让我觉得非常不环保。 米线是双人份调味料却只有一份，一个人煮时分量很尴尬。 虽然口味不算很油腻但是用完的锅和碗都会有挺厚的一层油边，我心理上过不去。 配料表相对来说已经算干净，可四个口味无一例外都是用的精炼植物油。 不过我把辣味的分给朋友之后对方光速吃完并且朝我索要链接，并且说比他买的另一款价格更高的松茸鸡汤米线好吃多了！ 安食六和兰州拉面 居然连拉州拉面都有速食装了！免煮，直接开水冲泡即可。确实是店里的味道，但杯面的分量超级少，完全吃不饱啊…… 但我发现这牌子有好多种包装，并且还有需水煮的袋装面，下次试试看。 拉面说 懒得找图了，一个曾经网红过的牌子，以料包丰盛出名，但不打折买的话完全不划算，而且除了番茄味的另外几款都好腻。 蓝色烟囱意面 又一个算是网红的牌子？朋友激情下单了全部口味但又不爱吃意面，最后几乎全投喂给我了。蟹黄味的咸且腻，其他的没什么记忆点。要是碰上打折可以囤点，起码不用自己开伙。 总结，还是老实自己做吧，我这个挑剔鬼真的不适合吃现成的！

一些偷懒的豆瓣书影音同步流程

喊了一万遍豆瓣已死和逃离豆瓣，可它还是简中世界最重要的书影音数据集中地。从 2008 年用豆瓣，这些年几乎每一本读过的书、看过的电影都在豆瓣标记过，其他数据无所谓，这长达十几年的档案还是割舍不下。最早豆瓣有封禁苗头时我陆续用 豆坟 导出过几次数据，但这只是一堆 excel 表格，静止的数据并没有价值。期间用了几天 Goodreads，也试图用 neodb 取代豆瓣，但豆瓣最重要的一点：关注友邻的精神食粮动态，目前仍没有对手。因此我在挣扎之下处于半弃瓣状态，仅用它做标记，不再发言。 现在我的书影音档案在四个地方：豆瓣、notion、blog 以及 neodb。 豆瓣 纯手动，读完/看完后挨个标记。 Notion Notion 同步用的是在长毛象看到的 教程 。 原理是通过 GitHub Action 定期抓取用户的豆瓣 rss feed 来获取更新，缺点是每次只能抓取最近 10 条记录，此前数据只能手动导入（用豆坟或油猴插件一次性获取）。而豆坟导出的数据不含海报信息，这样一来从前的记录无法显示图片，总觉得略微遗憾。notion 本身的显示页面也不是很灵活，加载还很慢，得不停点击 load more 才会继续显示，我单纯是因为不需要手动就姑且当多个备份仓库了。 实现成果如下图。 Blog 同样是全自动，只需初始设定完毕后续就无需手动操作。 教程 豆瓣书影音同步 GitHub Action 。 原理是使用 GitHub Action 定期抓取用户豆瓣书影音记录，在个人 GitHub 仓库里生成对应 csv 文件。优点是第一次启用就能获取全部历史数据，后续的自动更新也没有上面的十条数量限制，还能自行编辑 csv 文件增加新条目，比如被豆瓣和谐的那些。缺点是导出的数据居然缺了 ISBN！等我好好学习完 GitHub Action 就自己重写一个（遥遥无期的 flag ）。 但获取数据只是第一步，将数据可视化还需要更多折腾。作者提供了两个途径，一是同步到 blog，二是同步到 notion 。 同步到 blog 具体就是往自己的静态博客里加几个网页，再写对应的 CSS ，让 csv 文件能够以图片墙形式显示，我抄的这个 「电影 / 阅读」页面，顶！ ， 略微做了点改动。

随便写写

干了件非常离谱的事，晚上八点搭高铁去隔壁城市，在火车站出站口的地铁自助还书柜还了九本书，接着坐十分钟后的火车回我的城市。 原计划不是这样的。其实还有两本没看完，但已经逾期太久，本想着这两天冲刺一下，周五下午去还书，还完去附近我很喜欢的小店吃个饭，图书馆在周五六晚上会延迟开放到九点，还书后逾期记录消除，可以再接着在图书馆里挑几本借走，多么完美。 但今天情绪糟糕，三分生气，五分憋屈，总想着干点事发泄一下，看到案头堆着的这堆书，好吧，就你们了。 极限操作一番之后，身体很累，心情神奇地变好了一点。路上听 The First Slam Dunk 的 OST，正好听到良田海边回忆的那一段，忧伤又平静，成功安抚到了我。感激今年有这样的作品，可以让我在幻想的世界中逃避的久一些。

我选择保留偏见

混更新思路：翻开我的读书笔记。 几年前读特德姜的短篇集《降临》，《赏心悦目：审美干扰镜提案风波纪实》这篇我觉得很有意思，把当初的纸质笔记电子化一下存个档吧~ 总之就是基于人可以安装审美干扰镜去除外貌偏见的设定从而衍生出的一系列讨论。 读这篇时我反复摇摆，时而觉得去除审美干扰是个好主意，时而又觉得这相当于剥夺了一部分我的权利。 这篇的设置也非常有意思，多方人物多视角发言再加上不时的社会新闻播报，非常像现实生活中对热点事件的追踪。 而最终我的态度是：不会安装。 因为某种意义上，正是“我”的偏见、“我”的偏向和喜好塑造成了现在的“我”。哪怕一部分的我是由于当时的社会环境所造成，哪怕有失偏颇，但正是这所有的一切，才构成了当下的我。如果不能出于自由意志而做出决定，而是由科学、专家判定审美标准，又谈何自我呢？ 人是会变化的。人一生中会接触许许多多事物和理念，它们会从意想不到的方面去塑造人。就像流水击打卵石，恰似无所知觉，可每一次冲击都是雕琢的过程，水静止，卵石才定型。人也是这样吧，直到死亡的那一刻（或者停止思考的那一刻），才能盖棺论定，否则永远不要低估人改变的可能。 也正是这份流动性让每一个当下，每一份认知都变得独一无二。安装了审美干扰镜，相当于剥夺了人成长的一份乐趣。人会犯错，会走弯路，会先入为主，但是傲慢和偏见、嫉妒、自卑等等负面情绪造就了人的复杂性和迷人度，我不愿放弃这些。 而且，就算安装了外貌干扰镜又能如何呢？智商、性格、财富，等等等等，人类总会找到更多可判定、可比较的东西。更何况，谁又能保证这面镜子就是公正的？标准由谁制定？镜子又由谁制造？谁来安装，谁来调试，谁去后续跟踪维护数据？都是人，拥有偏见的人。因此永远不会存在绝对的公平。而在此基础上，让每个人都能根据自己喜好自由选择的社会，我觉得才是最大的公平。
